# PromptBits

Access your [PromptBits](https://promptbits.dev) prompt library directly from Raycast. Search, browse, and paste your saved prompts into any application with a single keystroke.

## Features

- **Quick Search** – Instantly search through your prompt library by name or description
- **One-Click Paste** – Paste prompts directly into any application
- **Variable Substitution** – Fill in dynamic placeholders before pasting prompts with arguments
- **Clipboard Copy** – Copy prompts to clipboard without pasting
- **Offline Caching** – Previously loaded prompts are cached for faster access

## Setup

1. Install the extension from the Raycast Store
2. Get your API key from [promptbits.dev](https://promptbits.dev)
3. Open Raycast and run the "Search Prompts" command
4. Enter your API key when prompted (or via Extension Preferences)

## Usage

1. Open Raycast and search for "Search Prompts"
2. Browse or search your prompt library
3. Press `Enter` to paste a prompt into the active application
4. Press `⌘ + C` to copy a prompt to clipboard
5. Press `⌘ + R` to refresh your prompts

### Prompts with Arguments

Some prompts contain dynamic variables (e.g., `{{name}}`). When you select one of these prompts, a form will appear to fill in the values before pasting.

## About PromptBits

[PromptBits](https://promptbits.dev) is a prompt management platform that helps you organize, store, and reuse your AI prompts across different tools and workflows.
